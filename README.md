**## Github 계정 만들기**

```
1. mkdir TIL
    1) TIL 파일 만들기
2. cd TIL
    1) TIL 파일로 이동
3. touch README.md
    1) README.md 파일 만들기
4. git init
    1) Git 저장소 생성하기
5. git add README.md
    1) working directory 상의 변경 내용을 staging area에 추가하기 위해 사용하는 Git 명령어
    2) git add . 를 사용하면 디렉토리 내 모든 파일을 add한다.
6. git commit -m "first commit"
    1) 커밋
7. git status
    1) 추가로 커밋할 것이 있는지 확인한다.
8. git branch -M master
    1) branch 명을 master로 지정한다.
9. git remote -v
    1) 로컬과 원격 연결
10. git log
    1) 커밋 변경사항을 추적할 수 있다.
11. git push -u origin master
    1) 로컬 -> 원격 저장소로 push
12. Github 사이트에 들어가서 New repositories를 통해 생성
13. settings 에서 Default branch 설정
```
